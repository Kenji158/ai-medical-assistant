# AI Medical Assistant

A simple AI-based medical assistant that suggests possible conditions based on user-reported symptoms.

## Features
- Voice and text symptom input
- Fuzzy matching for symptom recognition
- Suggests top 3 possible conditions
- Recommends appropriate medical specialists

## Technologies
- Python
- Flask
- SpeechRecognition
- pyttsx3
- FuzzyWuzzy

## Disclaimer
This project is for educational purposes only and does not replace professional medical advice.
